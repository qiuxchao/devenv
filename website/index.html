为开发者设计

不再一个一个工具切，不再一行一行改配置。

Profile 一键切换

为不同项目、不同场景创建独立配置方案，点击即切换整套环境变量。

可视化编辑

直观的表格界面，添加、修改、启用/禁用环境变量，无需记忆任何命令。

终端自动生效

一键注入 Shell Hook，新终端自动加载激活的环境配置，零手动操作。

代理快速切换

http_proxy、https_proxy 等代理变量按场景管理，公司/外网一键切。

跟随系统主题

自动适配明暗模式，也可手动切换。支持中英双语，跟随系统语言。

极致轻量

基于 Tauri 构建，安装包仅 ~5MB，内存占用远低于 Electron 应用。

工作原理

不魔改你的配置文件，不安装额外 CLI，简单透明。

1

在 GUI 中配置环境变量

创建 Profile，添加环境变量键值对，用开关控制启用/禁用。

2

激活 Profile，写入配置文件

点击 Activate 后，DevEnv 将变量写入 ~/.devenv/active.sh：

# ~/.devenv/active.sh
export JAVA_HOME='/opt/jdk-17'
export http_proxy='http://127.0.0.1:7890'
3

新终端自动加载

Shell 配置中的一行 source 命令让每个新终端自动读取配置：

# 自动注入到 ~/.zshrc
[ -f "~/.devenv/active.sh" ] && source "~/.devenv/active.sh"
